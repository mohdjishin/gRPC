# gRPC-Secure


### Running the Server

1. Navigate to the `cmd/server` directory:

    ```bash
    cd cmd/server
    ```

2. Run the server application:

    ```bash
    go run main.go
    ```

### Running the Client

1. Navigate to the `cmd/client` directory:

    ```bash
    cd cmd/client
    ```

2. Run the client application:

    ```bash
    go run main.go
    ```
